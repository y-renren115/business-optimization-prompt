# 個別学習

テスト結果などのデータを用いて、学生ごとに学習の進捗を確認し、個別の学習ニーズに合わせた学習支援を行います。

ベータ版の位置づけです。
正直いって、個別の学習者に最適な教材の作成が出来ているか不安です...

# Prompt

**ZoomIt**の**DemoType**用です。[end]までが1度のPromptです。

無料のZoomItのDemoTypeで、Prompt入力を簡単にする:

https://qiita.com/dahatake/items/0ba260742b70c7947de4


```cmd
# Step 0. ワークフロー立案
私は以下の{職務}をしており、以下の{業務}をしています。 
業務の一般的なワークフローを考えて詳細な説明をしてください。
- 職務: 高等学校の数学科と情報科の教員
- 業務: 学生ごとの個別学習の立案と実行
[end]
# Step 1. 小テスト作成
高等学校の{統計・データ分析のデータ活用}について、学生向けの個別学習のサポートを行います。
以下の{統計・データ分析のデータ活用}を参考にして、選択式の小テストの問題と、その回答と丁寧な回答の解説文を5問作成してください。

### 統計・データ分析のデータ活用
- 四分位偏差、分散、標準偏差:
	- これらの統計量の意味を理解し、それらを用いてデータの傾向を把握し、説明すること。
	- 四分位数、四分位範囲、四分位偏差、分散、標準偏差などの用語を知り、意味を理解させる。
	- 具体的な事象と関連付けて扱うことが大切であり、例えば、充電式機器の使用可能時間について、平均値や分散、標準偏差を求めて、その意味を理解させることが考えられる。

深呼吸をして、一歩ずつ進めてください。
[end]
# Step 2. 小テストの実施 (サンプルデータとして)
50人分の学生の{テスト結果}のサンプルデータを作成してください。1人の学生が全てのテスト問題に回答しています。
回答結果はランダムにしてください。

データには以下を含めてください。
- 日付: 全て同じ日付にします。
- 学生ID
- テスト問題の番号
- テスト問題の種類: 例 - 統計量の知識。四分位数の知識。具体的な事象との関連付け
- 学生の回答

最初の2名の学生の分を表形式で表示してください。
全ての採点結果をダウンロードできるようにしてください。

深呼吸をして、一歩ずつ進めてください。
[end]
# Step 3. 採点
作成した学生のテスト結果のサンプルデータ全て読み込んでください。
そして、{小テストの正解}を参照して、全ての学生の1問ずつ回答が、{小テストの回答}と同じかどうかの判定をしてください。
学生のテスト結果に、正解と、正誤判定の列が追加されたものが、採点となります。

採点結果のうち最初の2名の学生の分を表形式で表示してください。
全ての採点結果をダウンロードできるようにしてください。

深呼吸をして、一歩ずつ進めてください。
[end]
# Step 4. テスト結果の分析
{採点結果}のデータを分析思考を用いて、{統計・データ分析のデータ活用}に説明されている{目的}が達成できているかどうかの詳細な分析を行ってください。
適切なグラフを作成して、分析レポートを**日本語**で作成してください。
[end]
# Step 5. 学生ごとの分析
学生全員に同じではなく、学生ごとに{デ統計・データ分析のデータ活用}に説明されている{目的}が達成できているかの分析を行います。
**必ず**学生ごとに、テストの結果を評価をしてランク付けしてください。
その評価の結果から、学生ごと目的達成を助けてくれる、明確かつ具体的な学習目標を**学生ごと**に設定してください。
その学習目標の達成に役に立つ、これまでとは別の新しい選択式の小テストの問題と、その解説と詳細な解説を、一人一人の学生ごとに、学生ごとに最適なものを作成してください。**必ず**学生ごとに**別**にしてください。
学生ごとに、詳細かつ明瞭かつ具体的な日本語の分析のレポートを{分析レポートに含めるべき項目}を考慮して作成してください。

### 分析レポートに含めるべき項目
- 単元毎の点数の推移
- 推奨する目標
- 理解度の説明
- 教材: {統計・データ分析のデータ活用}を参考にした、新しい小テストの問題と、その回答と詳細な解説。
- フィードバック

最初の2人を表示してください。
全てのデータをダウンロードできるようにしてください。

深呼吸をして、一歩ずつ進めてください。
[end]
```
